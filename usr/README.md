# Poradnik NAFLD (Niealkoholowe Stłuszczenie Wątroby)

Aplikacja edukacyjna stworzona we Flutterze, pomagająca zrozumieć, leczyć i zarządzać niealkoholowym stłuszczeniem wątroby (NAFLD). Aplikacja zawiera porady dotyczące diety, stylu życia i polecanych ziół wspomagających regenerację wątroby.

## Funkcje

*   **Baza wiedzy:** Informacje na temat przyczyn i objawów stłuszczenia wątroby.
*   **Wskazówki dietetyczne:** Zalecenia żywieniowe wspierające zdrowie wątroby (np. dieta śródziemnomorska).
*   **Ziołolecznictwo:** Lista popularnych ziół, takich jak ostropest plamisty czy mniszek lekarski, wraz z opisem ich działania.
*   **Styl życia:** Porady dotyczące aktywności fizycznej i redukcji wagi.

## Technologie

*   Flutter (Dart)
*   Responsywny interfejs użytkownika dla urządzeń mobilnych, tabletów i środowisk webowych.

## Jak uruchomić

1.  Upewnij się, że masz zainstalowane środowisko Flutter.
2.  Sklonuj to repozytorium.
3.  Zainstaluj zależności uruchamiając komendę: `flutter pub get`.
4.  Uruchom aplikację za pomocą: `flutter run`.
